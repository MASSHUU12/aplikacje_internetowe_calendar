# Air pollution monitoring system

## Prerequisites

### Development w/ Docker

- [Docker](https://www.docker.com/products/docker-desktop/)
- [Bruno](https://www.usebruno.com/downloads) or other API client

### Development w/o Docker

- PHP 8.4^ (w/ pdo_mysql & zip extensions)
- Apache (or other) web server (via for example XAMPP)
- Composer
- Bun
- [Bruno](https://www.usebruno.com/downloads) or other API client
- Database (SQLite, MariaDB, etc.)
